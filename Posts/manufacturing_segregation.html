Two-Stage Perfume Manufacturing Workflow with Security Configuration

May 30, 2026 · 10 min read · Aaqib Bajwa

This document defines a standardized two-stage manufacturing workflow in Odoo v18, covering fragrance production, packaging, execution flow, and security configuration.

1. Overview

Production is divided into two phases: bulk fragrance mixing and finished goods packaging.

  • Phase 1: Fragrance Mixing (Bulk Production)
  • Phase 2: Packaging (Finished Goods)

2. Phase One: Fragrance Mixing

Product: FRAGRANCE MIX

  1. Create storable product
  2. Set UoM (KG / Liters)
  3. Create BoM (Mixing)
  4. Add raw materials

3. Phase Two: Packaging

Product: PERFUME 100ml

  1. Set product as finished goods
  2. Define BoM quantity
  3. Add FRAGRANCE MIX + packaging materials

4. Manufacturing Flow

Step 1

Create Manufacturing Order for finished product

Step 2

System generates bulk MO or create manually

Step 3

Produce bulk → then packaging → finish order

5. Security Rules

Restrict manufacturing orders to user ownership:

['|', ('user_id', '=', user.id), ('create_uid', '=', user.id)]
Always test record rules with non-admin users.
